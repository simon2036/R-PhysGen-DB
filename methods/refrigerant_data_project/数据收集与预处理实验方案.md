## 一、实验设计概述

### 1.1 研究目标

构建一个包含**多尺度物理信息**的制冷剂分子数据库，支持基于生成式 AI 的多目标分子设计优化。

### 1.2 数据尺度层次

根据论文要求，数据收集涵盖三个尺度：

| 尺度 | 数据类型 | 具体参数 | 来源 |
|------|---------|---------|------|
| **量子化学尺度（微观）** | 电子结构 | HOMO/LUMO 能级、能隙、偶极矩、极化率、结合能、电子亲和能、电离势 | QM9、PubChemQC、DFT 计算 |
| **分子动力学尺度（介观）** | 构象与热力学 | 分子构象能量、旋转势垒、振动频率、熵、焓、自由能 | MD 模拟、NIST WebBook |
| **状态方程尺度（宏观）** | 热物性 | 沸点、临界温度/压力/密度、饱和蒸气压、密度、粘度、热导率、COP | NIST REFPROP、ASHRAE |
| **环境与安全** | 环境指标 | GWP、ODP、大气寿命、可燃性等级、毒性、TFA 产率 | IPCC、EPA、文献 |

### 2.2 目标分子空间

根据论文中的制冷剂代际演化分析，收集以下类别：

1. **第一代 (CFCs)**: R-11, R-12, R-113, R-114, R-115 等（~15 种）
2. **第二代 (HCFCs)**: R-21, R-22, R-123, R-124, R-141b, R-142b 等（~20 种）
3. **第三代 (HFCs)**: R-32, R-125, R-134a, R-143a, R-152a, R-245fa, R-365mfc 等（~30 种）
4. **第四代 (HFOs)**: R-1234yf, R-1234ze(E/Z), R-1336mzz(Z), R-1233zd(E) 等（~15 种）
5. **天然工质**: R-744 (CO₂), R-717 (NH₃), R-290 (丙烷), R-600a (异丁烷) 等（~10 种）
6. **环状分子**: RC318 等（~5 种）

**总计**: ~100 种已知制冷剂分子 + 数据增强样本

### 2.3 数据收集流程

```python
# 数据收集伪代码
def collect_refrigerant_data():
    """
    多尺度数据收集流程
    """
    database = {}
    
    # 步骤 1: 从公共数据库获取基础数据
    for molecule in target_molecules:
        # NIST WebBook - 热力学数据
        thermo_data = fetch_from_nist(molecule.cas_number)
        
        # PubChem - 分子结构和描述符
        pubchem_data = fetch_from_pubchem(molecule.smiles)
        
        # QM9/PubChemQC - 量子化学数据
        quantum_data = fetch_quantum_data(molecule.smiles)
        
        # 整合数据
        database[molecule.id] = {
            **thermo_data,
            **pubchem_data,
            **quantum_data
        }
    
    # 步骤 2: 文献数据补充
    literature_data = extract_from_literature([
        'IPCC AR6',  # GWP 数据
        'ASHRAE Handbook',  # 热物性
        'NIST REFPROP',  # 状态方程参数
    ])
    
    # 步骤 3: 缺失数据计算
    for molecule in database:
        if missing_quantum_data(molecule):
            run_dft_calculation(molecule)  # DFT 计算
        
        if missing_thermo_data(molecule):
            run_md_simulation(molecule)  # MD 模拟
    
    return database
```

#### 3.1.2 异常值检测
使用**IQR 方法**识别异常值：
```python
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1

# 识别异常值
outliers = (data < Q1 - 3*IQR) | (data > Q3 + 3*IQR)

# Winsorization 处理
data_clean = data.clip(Q1 - 3*IQR, Q3 + 3*IQR)
```

#### 3.1.3 一致性检查
```python
# 检查 GWP 与制冷剂类型的一致性
```

#### 3.2.2 衍生特征

```python

df['efficiency_index'] = df['cop'] / (df['gwp_100yr'] + 1)

)

# 5. 量子化学衍生特征
```

### 3.3 缺失值处理

采用**多层次插补策略**：

```python
from sklearn.impute import KNNImputer

# 策略 1: 删除高缺失率特征 (>20%)
missing_ratio = df.isnull().sum() / len(df)
cols_to_drop = missing_ratio[missing_ratio > 0.2].index
df = df.drop(columns=cols_to_drop)

# 策略 2: KNN 插补数值型特征
imputer = KNNImputer(n_neighbors=5)
df[numeric_cols] = imputer.fit_transform(df[numeric_cols])

# 策略 3: 众数填充分类特征
for col in categorical_cols:
    df[col] = df[col].fillna(df[col].mode()[0])
```

对不同特征采用**差异化标准化策略**：

```python
def augment_data(df, n_augment=5):
    """
    数据增强：添加高斯噪声生成变体样本
    """
    augmented = []
    for _, row in df.iterrows():
        for i in range(n_augment):
            variant = row.copy()
            # 对数值特征添加小扰动
            for col in numeric_cols:
                variant[col] += np.random.normal(0, row[col] * 0.05)
            augmented.append(variant)
    return pd.DataFrame(augmented)
```

```python
class PhysicsInformedDiffusion(nn.Module):

        super().__init__()
        # 物理信息编码器
        self.quantum_encoder = nn.Linear(quantum_dim, hidden_dim)
```
