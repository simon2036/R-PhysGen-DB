## 5.1 GitHub 上传的大文件排除说明

本次上传前已检查 GitHub 不友好的大文件。`data/index/r_physgen_v2.duckdb` 在本地 residual rebuild 后约 `106 MiB`，属于可重建的查询索引快照，不是源数据表本身；为避免超过 GitHub 单文件限制和增加仓库体积，上传历史中排除此二进制增量。

- `data/gold/quality_report.json`：数据集版本、行数、coverage、quantum/cycle/active-learning 汇总。

- `data/bronze`、`data/silver`、`data/gold` Parquet 小表：可直接读取的分层数据资产。
- `docs/local_large_artifacts.md`：所有未上传大型 artifact / index 的用途、内容范围和重建方式。

## 7. 结论
