## 📋 方案概述

本方案针对论文《多目标约束下的制冷剂分子设计：从代际演化到 AI 驱动范式》附录中**方向一**提出的数据需求，设计了一套完整的数据收集与预处理实验流程。

### 核心目标

构建支持**多目标生成模型优化**的制冷剂分子数据库，包含：
- ✅ 量子化学尺度数据（微观）
- ✅ 分子动力学尺度数据（介观）
- ✅ 状态方程尺度数据（宏观）
- ✅ 环境与安全指标

## 📊 数据预处理流程

### 步骤 1: 数据清洗

```python
# 1.1 去重处理
df = df.drop_duplicates(subset=['smiles'], keep='first')

# 1.2 异常值检测 (IQR 方法)
Q1 = data.quantile(0.25)
Q3 = data.quantile(0.75)
IQR = Q3 - Q1
data = data.clip(Q1 - 3*IQR, Q3 + 3*IQR)

# 1.3 一致性检查
# 验证 HFOs 的 GWP < 10
# 验证 CFCs 的 ODP > 0.5
```

### 步骤 2: 特征工程

#### 2.1 RDKit 分子描述符
- 分子量 (MolWt)
- 疏水性 (LogP)
- 极性表面积 (TPSA)
- 原子数
- 环数
- QED 评分

#### 2.2 衍生特征（基于论文多目标优化需求）

```python
# 能量效率指标
efficiency_index = COP / (GWP + 1)

# 环境风险综合指标
environmental_risk = GWP/1000 + ODP*100 + lifetime/50

# 安全性评分
safety_score = map(ashrae_safety)  # A1→1, A2L→2, ..., B3→7

# 综合性能指标
comprehensive_score = COP/5.0 - GWP/5000 - safety_score/7.0

# 量子化学衍生
electrophilicity = (HOMO + LUMO)² / (8 × gap)
chemical_hardness = gap / 2
electronegativity = -(HOMO + LUMO) / 2
```

### 步骤 3: 缺失值处理

采用**多层次插补策略**：

| 缺失率 | 处理方法 |
|--------|----------|
| >20% | 删除该特征 |
| 5-20% | KNN 插补 (k=5) |
| <5% | 均值/众数填充 |

### 步骤 4: 特征标准化

差异化标准化策略：

```python
# 物理量：Min-Max 归一化 [0, 1]
physical_features → MinMaxScaler()

# 能量相关：Z-score 标准化
energy_features → StandardScaler()

# 环境指标：对数变换
GWP, lifetime → log1p()

# 分类变量：One-hot 编码
type, ashrae_safety → get_dummies()
```

### 步骤 5: 数据集划分

```python
# 分层抽样确保化学空间均匀覆盖
train:val:test = 70%:15%:15%
stratify by refrigerant_type
```

### 物理信息嵌入机制

根据论文建议，在生成模型中嵌入多尺度物理信息：

```python
class PhysicsInformedGenerator:
    """
    物理信息嵌入的生成模型
    基于论文 6.3 节：物理信息嵌入机制
    """
    def __init__(self):
        # 多尺度物理信息编码器
        self.quantum_encoder = QuantumEncoder()  # 量子化学
        self.thermo_encoder = ThermoEncoder()    # 热力学
        self.env_encoder = EnvEncoder()          # 环境指标
        
        # 生成网络 (扩散模型/VAE)
        self.generator = DiffusionModel()
    
    def generate(self, conditions):
        # 嵌入物理约束
        physics_emb = self.encode_physics(conditions)
        
        # 条件生成
        molecules = self.generator.sample(conditions, physics_emb)
        
        return molecules
```

## 📝 关键实施要点

### 1. 数据来源优先级

```
第一优先级 (高可信度):
├── NIST Chemistry WebBook
├── NIST REFPROP
├── IPCC Assessment Reports
└── ASHRAE Standards

第二优先级 (文献数据):
├── 同行评审论文
├── 技术报告
└── 专利数据

第三优先级 (计算数据):
├── DFT 计算 (B3LYP/6-311+G(d,p))
└── MD 模拟
```

### 2. 计算参数配置

```yaml
# DFT 计算参数
quantum_chemistry:
  method: DFT
  functional: B3LYP
  basis_set: 6-311+G(d,p)
  properties:
    - HOMO_energy
    - LUMO_energy
    - dipole_moment
    - polarizability
    - binding_energy

# MD 模拟参数
molecular_dynamics:
  force_field: OPLS-AA
  ensemble: NPT
  temperature: 298.15 K
  pressure: 1 atm
  simulation_time: 10 ns
```

### 3. 质量控制流程

```
原始数据 → 去重 → 异常值检测 → 一致性检查 → 特征工程 → 
缺失值处理 → 标准化 → 数据集划分 → 质量评估 → 最终数据集
```
